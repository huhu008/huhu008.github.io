网易笔试java编程

发表于 2017-09-10





还是只会第一题。。。。


问题大意：



有两种兔子黑兔和白兔，黑兔第一年生3只兔子，第二年死亡。白兔第一年生2只，第二年生三只，第三年死亡。
现有黑兔白兔各N只，过了m年后，两种兔子数量相差多少？

要求：



输入一个数字N，再输入一个数字M年，输出两种兔子的差Q。

例如：



输入数字N=5,M=3，输出数字Q=35.

解题思路：



黑兔：可以通过创建变量j1，j2，j1表示新出生的黑兔，j2表示第二年的黑兔。
白兔：因为白兔可存活3年，所以需要创建4个变量，k k1 k2 k3.来其中变量k为暂存k2变量，k1表示新出生的白兔，k2表示第二年的白兔，k2表示第三年的白兔。

我的代码：




import java.util.;

public class Nothing2 {
    public static void main (String[] args){
        Scanner a = new Scanner(System.in);
        int n = a.nextInt();
        Scanner b =new Scanner(System.in);
        int years = b.nextInt();
        int black=0;
        int white=0;
        int m=0;
        int j1=n;
        int j2=0;
        int k1=n;
        int k2=0;
        int k3=0;
        int k=0;
        for(int i=0;i<years;i++){

        //求出第i年黑兔的数量
            j2=j1;
            j1=3j1;
            black=j1+j2;
            System.out.println(black);


        //求出第i年白兔的数量
            k3=k2;
            k=k2;
            k2=k1;
            k1=3k+2k1;
            white=k1+k2+k3;
            System.out.println(white);


    }

    //m年后黑兔白兔的差。
        m=black-white;

        System.out.println(m);
    }

}
