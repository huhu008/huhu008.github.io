ZTE笔试java编程

发表于 2017-09-09





笔试一共两道java编程，只会第一题先记录一下。


问题大意：



小明有一个硬币，投入A机器产出2N+1个硬币。投入B机器产出2N+2个硬币。要刚好产出M个硬币，如何使用机器？

要丢：



输入一个数字M，将硬币投入A中输出1，投入B中输出2。最终输出一串数字如：“11222112。。。。”

例如：



输入数字10，输出数字串“122”.

解题思路：



通过画图不难发现这是一个类似金字塔的数字排列顺序。第N次投入所有硬币,会有2^N个可能结果。我们只需将输入数字M在本层中所处的位置找出，再不断向上查询，就可得到最终的投入方法。

我的代码：




import java.util.Scanner;

public class Noting {

        public static void main(String[] args) {
            Scanner a = new Scanner(System.in);
            int n = a.nextInt();
            int m=0;
            for(int i=0;i<100;i++){
                if(n<Math.pow(2,i)-1){
                    m=i;
                    break;
                }
            }
            //本行数组共有 num 个数字。
            int num=(int)Math.pow(2,m-1);

            int min=(int)Math.pow(2,m-1)-1;
            int max=(int)Math.pow(2,m)-2;
            int q=0;
            q=n-(min-1);
            for(int j=0;j<m-1;j++){
                //当数字M在本行数组处于左半部时输出1
                if(q<=num/2){
                    System.out.print(“1”);
                    num=num/2;

                }
                else{
                //当数字M在本行数组处于右半部时输出2
                    System.out.print(“2”);
                    q=q-num/2;
                    num=num/2;
                }
            }

        }

}
